# Simon Game

### **Game Overview:**
- **Objective:** The player must replicate a sequence of button presses that the game displays. The sequence gets progressively longer with each successful replication.

### **How to Play:**
1. **Start the Game:** Press any key to begin. The game will start with an introductory sequence of button presses.
2. **Game Sequence:** The game randomly selects a button and flashes it. This sequence is displayed to the player.
3. **Player Input:** The player must click the buttons in the same order as the game’s sequence.
4. **Correct Sequence:** If the player correctly replicates the sequence, the game moves to the next level, adding another button to the sequence.
5. **Incorrect Sequence:** If the player makes a mistake, the game ends, displaying the player’s score (which is the current level).
