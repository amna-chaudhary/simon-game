# Simon Game 🎮

## 📌 Description

The Simon Game is a classic memory-based game where players must replicate an increasingly complex sequence of button presses. This project demonstrates interactive game logic, event handling, and user input validation.

## 🎯 Objective

The goal of the game is to correctly repeat the sequence generated by the system. With each successful attempt, the sequence becomes longer, increasing the difficulty and testing the player's memory.

## ⚙️ How It Works

* The game generates a random sequence of button flashes.
* The player observes and memorizes the sequence.
* The player must reproduce the exact sequence by clicking the buttons in the correct order.
* After each correct round, a new step is added to the sequence.
* The game continues until the player makes an incorrect move.

## 🕹️ How to Play

1. Press any key to start the game.
2. Watch the sequence of flashing buttons carefully.
3. Repeat the sequence by clicking the buttons in the same order.
4. Progress through levels as the sequence grows.
5. The game ends when an incorrect input is made.

## 🚀 Features

* Interactive and responsive gameplay
* Increasing difficulty with each level
* Real-time user input validation
* Score tracking based on levels
* Visual feedback for correct and incorrect inputs
